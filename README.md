 # 👥 Employees Insight Dashboard (HR Analytics)

## 📊 Overview
A dynamic **Power BI Dashboard** designed to analyze Human Resources data and provide actionable insights into workforce distribution, financial performance, and employee retention.

---

## 🎯 Key Objectives

- 📌 Monitor workforce status (Active vs Inactive employees)
- 📉 Analyze employee attrition (Voluntary vs Involuntary)
- 💰 Evaluate salary distribution and total workforce cost
- 📈 Track employee growth and sick leave trends over time
- 👥 Analyze demographics (age, gender, and business units)

## 🛠️ Tech Stack

- 📊 Microsoft Power BI → Data visualization & dashboard design  
- 🧮 DAX → Calculated measures (tenure, cumulative KPIs, averages)  
- 🔄 Power Query → Data cleaning and transformation  

---

## 📂 Project Purpose
This dashboard helps HR teams make data-driven decisions to improve employee retention, optimize workforce costs, and understand workforce structure.
